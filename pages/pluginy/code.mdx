# HTML Kód

Plugin HTML kód je ideální pro jakýkoli kód, pro který například není vytvořený plugin. Obsah není omezený ničím, lze použít jakýkoli kód který je kompatibilní se standardy HTML (vč. Javascript a CSS)

<img src="/code/button-code.png" alt="Kód tlačítko" style={{ float: "left", width: 30, marginRight: 5 }} /> Pro vložení HTML bloku stiskněte tlačítko `Kód` 

to vloží prázdný box s ikonami na pravé straně 
<img src="/code/empty.png" alt="Accordion" /> 

<img src="/code/btn-edit.png" alt="tlačítko upravit" style={{ float: "left", width: 30, marginRight: 5 }} /> 
Abychom mohli vložit HTML kód, stlačte tlačítko pro úpravu. Po kliknutí na tlačítko se objeví pole kam kód vložíme.
